# VSCode Logger

Welcome to the VSCode Logger extension documentation. The extension streams logs from embedded Linux devices over SSH, colourizes entries inside a Webview, and lets you save presets or export filtered output.

## Contents

- [Extension Guide](extension-overview.md) for a tour of the extension experience.
- [API Reference](api/index.md) generated from the TypeScript sources using Doxygen and Doxybook2.

## Getting Started

1. Install dependencies with `npm install`.
2. Run `npm run compile` to build the extension output.
3. Press **F5** in VS Code to launch a development host with the **Embedded Logger** view.

For usage instructions and settings details, see the [Extension Guide](extension-overview.md).
