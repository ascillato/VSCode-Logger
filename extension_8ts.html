◆ validateSshDevice()

function validateSshDevice (   device:EmbeddedDevice)

Migrates legacy passwords into VS Code SecretStorage.

Users might still have passwords stored in their settings for convenience. This function copies those values into SecretStorage so future connections can retrieve them securely without modifying the user's settings.json.

Parameters
contextThe extension context used to access SecretStorage.
devicesThe list of configured devices whose passwords need migration.
