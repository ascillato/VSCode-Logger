Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue. Instead, please report it by emailing secur.nosp@m.ity@.nosp@m.scall.nosp@m.ant..nosp@m.com. We will be highly responsive to all security concerns and ask that you give us sufficient time to investigate and address the vulnerability before disclosing it publicly.

Please include the following details in your report: 

A description of the vulnerability
Steps to reproduce the issue
Your assessment of the potential impact
Any possible mitigations
